# Site J&D Servicos Veterinarios

Este projeto e um site estatico simples para publicar no GitHub Pages.

## Como visualizar localmente

- Abra o arquivo `index.html` no navegador.

## Publicar no GitHub Pages

1. Crie um repositorio no GitHub (ex: `site-jd`).
2. Suba a pasta `site-jd` inteira para o repositorio.
3. No GitHub, vÃ¡ em Settings > Pages.
4. Em "Branch", selecione `main` e a pasta `/ (root)`.
5. Salve e aguarde a URL do GitHub Pages.

## Apontar dominio do UOL

- Crie um arquivo `CNAME` no repositorio com seu dominio (ex: `jdservicos.com.br`).
- No UOL, aponte o DNS para o GitHub Pages (registros `A` e `CNAME`).
